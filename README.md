# HAHA_2026

Frontend для интеллектуальной платформы агрегации, кластеризации и анализа новостей с интерактивной визуализацией сущностей и связей.
Кейс "Фонда развития IT образования".

## Настройка окружения

Создай файл `.env` в корне проекта:

```
VITE_API_URL=http://localhost:8000
```


## Запуск проекта

Установи зависимости:

```
npm install
```

Запусти проект в режиме разработки:

```
npm run dev
```


## Сборка

Собери проект:

```
npm run build
```

Запусти предпросмотр сборки:

```
npm run preview
```
